# Concentration Game

Concentration is a matching card game. For the player to win, the user needs to match all the cards by matching 2 of the same icon type. Built with [React JS](https://reactjs.org).

## Features
  1. Sound effects
  2. Set volume level
  3. Mute sound
  4. Save time
  5. Leaderboard
  6. Change card icon type
